LoRa Sensor Network in Solid Waste Collection

17 June 2022

Mass production and population growth have led to increased municipal solid waste (MSW) in urban areas, challenging waste collection systems. Traditional methods rely on fixed routes, which are inefficient and don't separate recyclable materials. This project prototypes a sensing system that optimizes waste collection by monitoring dumpster fill levels using ultrasonic sensors and LPWAN (LoRa) technology.

Event-Driven Architecture

15 August 2024

Event-driven architecture is a modern software design pattern that allows different applications, devices, or systems to communicate in a loosely coupled way by using events. In this context, an "event" is any significant change or occurrence that an application or system can detect, like a customer placing an order or a sensor recording a temperature change.
